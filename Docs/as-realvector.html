Detailed Description

The RealVector class is a class that provides an array of the QVector<qreal> type, which is extended with additional functions.

bool RealVector::isZero() const

Returns true if all the vector elements are 0; otherwise returns false.

Example:


  // vector: [-3.0, 0.0, 3.0]
  // vector.isZero(): false

qreal RealVector::max() const

Returns the largest element of the vector.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.max(): 5.0

qreal RealVector::mean() const

Returns the mean (average) of the vector elements.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.mean(): 3.0

qreal RealVector::middle() const

Returns the middle point between the min and max vector elements.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.middle(): 2.0

qreal RealVector::min() const

Returns the smallest element of the vector.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.min(): 1.0

RealVector RealVector::normalizeBy(const As::RealVector &other) const

Returns a vector with elements divided by the respective elements of the other vector.

Example:


  // vector: [4.0, 9.0, 100.0]
  // other:  [2.0, 2.0,  25.0]
  // vector.normalizeBy(other): [2.0, 4.5, 4.0]

qreal RealVector::range() const

Returns the range of the vector elements.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.range(): 4.0

RealVector RealVector::reverse() const

Returns a vector with a reversed order of elements.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.reverse(): [1.0, 5.0, 3.0]

RealVector RealVector::simplified() const

Returns a vector with just a single element if all the vector elements are equal; otherwise returns the original vector.

Example:


  // vector1: [3.0, 3.0, 3.0]
  // vector1.simplified(): [3.0]
  // vector2: [0.0, 3.0, 3.0]
  // vector2.simplified(): [0.0, 3.0, 3.0]

const RealVector RealVector::sqrt() const

Returns a vector with elements equal to square root of the original vector elements.

Example:


  // vector: [4.0, 9.0, 100.0]
  // vector.sqrt(): [2.0, 3.0, 10.0]

qreal RealVector::step() const

Returns the mean (average) step between the vector elements.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.step(): 2.0

qreal RealVector::sum() const

Returns the sum of the vector elements.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.sum(): 3.0 + 5.0 + 1.0 = 9.0

qreal RealVector::sumSqr() const

Returns the sum of the squared vector elements.

Example:


  // vector: [3.0, 5.0, 1.0]
  // vector.sumSqr(): 9.0 + 25.0 + 1.0 = 35.0
