Detailed Description

The StringParser is a class based on the As::String that provides a string parser.

int StringParser::indexOfText(const QString &text, const QStringList &list, const int from = 0) const

Returns the index position of the first exact match of text in the list, searching forward from index position from. Returns -1 if no item matched.

By default, this function is case sensitive.
