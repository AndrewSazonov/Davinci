Detailed Description

The RealArray class is a base class that provides an array of the QVector<qreal> type, which takes just the required functions of QVector.

bool RealArray::isEmpty() const

Returns true if the base array is empty; otherwise returns false.

QVector<qreal> RealArray::mid(const int pos, const int length = -1) const

Returns a sub-vector which contains elements from this vector, starting at position pos. If length is -1 (the default), all elements after pos are included; otherwise length elements (or all remaining elements if there are less than length elements) are included.

void RealArray::prepend(const qreal value)

Inserts value at the beginning of the array.

[protected] void RealArray::resize(const int size)

Sets the size of the array to size. If size is greater than the current size, elements are added to the end; the new elements are initialized with a default-constructed value. If size is less than the current size, elements are removed from the end.

[virtual] void RealArray::set(const QVector<qreal> &other)

Sets the array with other.

int RealArray::size() const

Returns the number of elements in the array.

Example:


  // array: [3.0, 5.0, 1.0]
  // array.size(): 3

bool RealArray::operator==(const As::RealArray &other) const

Returns true if other is equal to this array; otherwise returns false. Two arrays are considered equal if they contain the same values in the same order.

qreal &RealArray::operator[](const int i)

Returns the element at index position i in the base array.

i must be a valid index position in the array (i.e., 0 <= i < size()).

Example:


  // array: [3.0, 5.0, 1.0]
  // array[0]: 3.0
