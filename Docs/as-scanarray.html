Detailed Description

The ScanArray is a class that provides an array of the measured scans.

const As::RealVector ScanArray::xyzToAngles(const qreal wavelength, const qreal x, const qreal y, const qreal z) const

Returns the calculated scattering angles from the given wavelength wavelength and reciprocal lattice vectors x, y, z. Angles are returned in the form of As::RealVector with the following order: 0 - twotheta, 1 - omega, 2 - chi, 3 - phi (Four-circle geometry), 4 - twotheta, 5 - gamma, 6 - nu, 7 - omega (Lifting counter geometry).

const As::RealVector ScanArray::xyzToHkl(const As::RealMatrix9 &ub, const qreal x, const qreal y, const qreal z) const

Returns the calculated Miller indices h, k, l from the given UB matrix ub and reciprocal lattice vectors x, y, z. Miller indices are returned in the form of As::RealVector with the following order: 0 - h, 1 - k, 2 - l.
