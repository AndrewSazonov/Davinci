Detailed Description

The Line class provides a geometrical line build by 2 points with given coordinates.

Member Function Documentation

Line::Line()

Constructs a default line.

Line::Line(const qreal x1, const qreal y1, const qreal x2, const qreal y2, const qreal esdX1 = 0., const qreal esdY1 = 0., const qreal esdX2 = 0., const qreal esdY2 = 0.)

Constructs a line based on the coordinates of two points (x1, y1) and (x2, y2). The respective estimated standard deviation (ESD) of the coordinates (esdX1, esdY1) and (esdX2, esdY2) should also be given if differ from 0.

[virtual] Line::~Line()

Destroys the array.

qreal Line::esdXForY(const qreal y, const qreal esdY) const

Returns the calculated x ESD for the given y coordinate and its esdY ESD.

qreal Line::esdYForY(const qreal y) const

Returns the calculated y ESD for the given y coordinate.

qreal Line::xForY(const qreal y) const

Returns the calculated x coordinate for the given y coordinate.
